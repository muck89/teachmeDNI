Let's check it out in GTM!

Last but not least, lets talk about GTM (Google Tage Manager) and Google Tag Assistant.

GTM is a great way to add the JS to a site if the agency/client does not have access to editing the site. All that needs to be done is to add the script in a GTM container. However, make sure you advise your client that they will have to choose either adding the JS in the head of the site or in GTM. It can not be placed in both simultaneously because it will cause a misfire in the code and it will not work properly.

How to check if GTM is applied correctly or not.

When a client is having trouble with DNI using GTM it is crucial to make sure that the JS code is matching what was put in a GTM container. However, most of the time client may not want to give us access to the GTM account and if we do get access we only will have a view option. That is where Google Tag Assistant comes in. This chrome extension is very useful. It will advise if a code in the applied container is firing correctly or not. The extension is free and in the google app store. To see if the correct JS is applied you will need to check in CFA for the DNI record code, it is the long code with numbers and letters after getDNIRecord ("code").
